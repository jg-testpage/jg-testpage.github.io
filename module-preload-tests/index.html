This is a test page for preloading ES modules via modulepreload, preload, or both

The goal is to compare whether those methods work, and if they don't result in double downloads

Note that link rel=preload is not really meant for ES modules; see why doesn't link rel="preload" work for modules

The links below do not test cross-domain scripts but same-origin ones.
